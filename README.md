**Roblox Unblocker v1.0**
===================

For Windows:
1. Run install.bat
2. Use run_region.bat or region_win.bat

For Linux:
1. Run: chmod +x install.sh
2. Run: ./install.sh
3. Use run_region.sh or region_linux.sh

Commands:
- set: Apply random region
- restore: Restore original settings
- status: Show current settings

Note: Linux requires sudo for timezone changes.
